任何一个容器可以指定 display: flex

行内元素 display: inline-flex

设置为Flex 后， 子元素的 float clear vertical-algin 将失效

基本概念 容器 项目

容器的属性

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

项目的属性

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self
