运行项目

`npm init;npm start;`

`打开localhost:3000`


如果想自己从头敲一遍

1. 先安装react脚手架
`npm install -g create-react-app`

2. 用create-react-app创建自己的项目
`create-react-app my-react-demo(项目文件名)`

3. 等待安装完成，启动项目`cd my-react-demo; npm start`

4. 打开浏览器输入`localhost:3000` 默认会自己打开

__国内可能会因为网速不好npm安装插件失败,可改用cnpm__
